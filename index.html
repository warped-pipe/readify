What is readify?

readify provides a safe, easy, open-source solution that allows you to read articles in without annoying ads and popups.

How do I use it?

It's incredibly easy! Simply visit any article on a modern web browser and place readify.warp-pipe.net/ before the URL.

Example:
readify.warp-pipe.net/https://www.bbc.co.uk/news/newsbeat-65791039

Chrome user?

Looking for an even easier way to toggle between an article and readify? Install the readify extension on the Chrome Web Store.

Firefox user?

Looking for an even easier way to toggle between an article and readify? Install the readify extension on the Firefox Extensions Store.
